# AR-Block-Builder
Seminaarityöni ohjelmistokehityksen teknologiat -kurssille.

Jos haluat testata omalla puhelimellasi tätä, niin asenna Android-kansion sisällä oleva tiedosto nimeltä Builder-1.apk.
